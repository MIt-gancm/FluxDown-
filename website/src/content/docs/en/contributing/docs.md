These docs live in the public [zerx-lab/fluxdown](https://github.com/zerx-lab/fluxdown) repository — every page has an **Edit this page** link.

## How it works

1. Click *Edit this page* — GitHub will fork the repo for you.
2. Edit the Markdown and open a pull request.
3. A maintainer reviews and merges it.

Prefer not to use Git? Use the feedback form at the bottom of any page.
